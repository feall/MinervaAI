# MinervaAI

## Visão geral

Este projeto auxilia estudantes que fizeram o ENEM a explorarem e escolherem cursos de graduação na **UFRJ**, utilizando dados históricos de admissões e similaridade entre grades curriculares.

O sistema permite:

* 🎯 Estimar a probabilidade de aprovação em diferentes cursos com base em dados históricos de listas de espera e notas de corte
* 📊 Recomendar cursos semelhantes com base na estrutura curricular e disciplinas em comum
* 🧭 Ordenar cursos de acordo com maior compatibilidade com o perfil do estudante

---

## Por que este projeto existe

A escolha de um curso universitário envolve muitas incertezas e, muitas vezes, pouca base em dados reais.

Este projeto busca tornar essa decisão mais informada ao:

* Utilizar dados históricos reais de ingresso
* Reduzir a incerteza na escolha de cursos
* Apresentar alternativas de cursos que o estudante talvez não considerasse
